# Anomaly detection

Develop an autoencoder model for credit card fraud detection and deploy it as a serverless AWS inference service.

# Data sources

This project uses the Credit Card Fraud Detection dataset:

https://www.openml.org/search?type=data&sort=runs&id=42175&status=active
