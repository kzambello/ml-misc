# Wind forecast

Forecast offshore wind power production using ARIMA and LSTM models.

# Data sources

This project uses the EMHIRES dataset:

https://data.jrc.ec.europa.eu/dataset/jrc-emhires-solar-generation-time-series

Relevant paper

I. Gonzalez Aparicio, A. Zucker, F. Careri, F. Monforti-Ferrario, T. Huld, J. Badger, 'EMHIRES dataset Part I: Wind power generation. European Meteorological derived HIgh resolution RES generation time series for present and future scenarios', Publications Office of the European Union, JRC103442 (2016)
