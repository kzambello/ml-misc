# ml-misc

Repository of miscellaneous ml projects.

1. **Glass identification**:

   Compare different classification algorithms for forensic glass identification.

2. **Energy efficiency**:

   Compare different regression models for energy efficiency prediction.

3. **Face recognition**:

   Develop a convolutional neural network (CNN) for face recognition that performs multi-attribute classification (gender, head-pose and eye-occlusion).

4. **Wind forecast**:

   Forecast offshore wind power production using ARIMA and LSTM models.

5. **Exoplanet**:

   Answer natural-language questions about NASA's exoplanet dataset using flan-t5 and either a simple RAG system or LoRA fine-tuning.

6. **Anomaly detection**:

   Develop an autoencoder model for credit card fraud detection and deploy it as a serverless AWS inference service.
