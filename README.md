# ml-misc

Repository of miscellaneous ml projects.

1. **Glass identification**:

   Compare different classification algorithms for forensic glass identification.

2. **Energy efficiency**:

   Compare different regression models for energy efficiency prediction.

3. **Face recognition**:

   Develop a convolutional neural network (CNN) for face recognition that performs gender classification (male, female), head-pose classification (left, right, straight, up) and eye-occlusion detection (wearing sunglasses, not wearing sunglasses).

4. **Wind forecast**:

   Forecast offshore wind power production using ARIMA and LSTM models.
